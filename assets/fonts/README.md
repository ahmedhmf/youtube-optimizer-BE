# Fonts for YouTube Thumbnails

## Required Fonts

To generate professional YouTube thumbnails, you need to add bold, eye-catching fonts to this directory.

### Recommended Free Fonts:

1. **Montserrat Bold** (Most Popular for Thumbnails)
   - Download: https://fonts.google.com/specimen/Montserrat
   - Files needed: `Montserrat-Bold.ttf`, `Montserrat-ExtraBold.ttf`

2. **Bebas Neue** (Great for Impact)
   - Download: https://fonts.google.com/specimen/Bebas+Neue
   - File needed: `BebasNeue-Regular.ttf`

3. **Anton** (Bold & Condensed)
   - Download: https://fonts.google.com/specimen/Anton
   - File needed: `Anton-Regular.ttf`

4. **Oswald Bold** (YouTube Favorite)
   - Download: https://fonts.google.com/specimen/Oswald
   - File needed: `Oswald-Bold.ttf`

### How to Add Fonts:

1. Download the font files from Google Fonts
2. Extract the `.ttf` files
3. Place them in this `assets/fonts/` directory
4. Restart the application

### Currently Using:
The system will use system fonts as fallback until you add custom fonts here.
